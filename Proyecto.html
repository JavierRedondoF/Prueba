En la actualidad los programadores web basan sus creaciones en miles de estructuras de códigos entre esos HTML, uno de los lenguajes más utilizados a nivel mundial para la creación de sitios web. HTML es el acrónimo para "Hypertext markup language" o "lenguaje de marca de hipertexto", el cual se utiliza para estructurar, desplegar y crear una página web junto a sus contenidos multimedia por medio de etiquetas, entre sus características más relevantes son: insertar párrafos, listas, videos, imágenes, tablas de datos y otros contenidos que aportan dinamismo al sitio.
